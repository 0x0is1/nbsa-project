# Naive Bayes Sentiment analysis

### Dataset used
https://www.kaggle.com/datasets/parulpandey/emotion-dataset

### Install and run
```
git clone https://github.com/0x0is1/nbsa-project
cd nbsa-project
pip install pandas nltk
python test_nbsa.py
```

### Requirements
* nltk
* pandas

### Maximum accuracy achieved: 0.529
